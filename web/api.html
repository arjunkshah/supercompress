API reference

Hosted stack with our Tavily, Composio, and Nebius keys. You need a SuperCompress API key only.

POST /v1/agent/turn — the product

Send a query. We research (Tavily), gather apps (Composio), compress (SuperCompress), answer (Nebius).

POST https://supercompress-api.onrender.com/v1/agent/turn
Content-Type: application/json
X-API-Key: sc_live_YOUR_KEY

{ "query": "What PRs need review and what's blocking ship?" }

→ {
  "answer": "...",
  "memory": { "kv_savings_pct": 65.0, "original_tokens": 2400, "kept_tokens": 840 },
  "phases": [
    { "phase": "tavily", "detail": "6 sources" },
    { "phase": "composio", "detail": "GitHub · Gmail · Linear" },
    { "phase": "memory", "detail": "65% KV savings" },
    { "phase": "nebius", "detail": "..." }
  ],
  "actions": []
}

What runs under the hood

Tavily search→ Composio gather→ SuperCompress→ Nebius + tools

Optional flags: search_web, gather_apps (both default true).

Advanced: POST /v1/compress/blocks

If you already gather context yourself and only need compression. Most apps should use /v1/agent/turn.
