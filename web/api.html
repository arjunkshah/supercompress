Developer API

Drop into any agent loop — Python package or HTTP from any language.

Option A — pip install (Python)

pip install git+https://github.com/arjunkshah/supercompress.git

from supercompress import compress_for_turn
compressed, stats = compress_for_turn(blocks, query)
# send `compressed` to your LLM

Option B — HTTP API (any language)

Run locally: ./bin/supercompress serve then POST JSON.

POST http://127.0.0.1:8787/v1/compress/blocks
Content-Type: application/json

{
  "context_blocks": ["## Tavily\\n...", "## GitHub\\n..."],
  "query": "What should I ship today?",
  "budget_ratio": 0.35
}

→ { "compressed_text": "...", "stats": { "kv_savings_pct": 65.2 } }

Live docs when server is running: 127.0.0.1:8787/docs

Where it goes in your loop

Gather→ POST /v1/compress/blocks→ LLM→ Tools
