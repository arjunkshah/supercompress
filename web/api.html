API reference

Every call runs Tavily, Composio, SuperCompress, and Nebius.

Primary

POST /v1/agent/turn

Send a query + optional app context. Get an answer from Nebius.

curl -X POST /v1/agent/turn \
  -H "X-API-Key: sc_live_YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "What should I ship today?",
    "context_blocks": [
      "## Tasks\n- Onboarding flow",
      "## Reminders\n- Standup 9am"
    ]
  }'

Response

{
  "answer": "...",
  "memory": { "kv_savings_pct": 65.0, "original_tokens": 2400, "kept_tokens": 840 },
  "stack": { "tavily": true, "composio": true, "supercompress": true, "nebius": true },
  "phases": [...],
  "actions": []
}
